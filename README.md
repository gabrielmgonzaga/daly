# Daly

A simple weather search to convert between Fahrenheit and Celsius temperatures.

### Technologies:
- HTML5
- CSS3
- Bootstrap 4
- vanilla Javascript
- Node JS
- Express JS
- Open Weather API

### Features:
- Display weather data for global cities.
- View temperatures in Fahrenheit or Celsius.
